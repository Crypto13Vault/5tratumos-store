# 5tratumOS App Store

Custom app store for [5tratumOS](https://5tratumos.ai).

## Apps

- **Bitaxe Dashboard** — Web UI for monitoring & managing Bitaxe ASIC miners

## Adding to 5tratumOS

1. Open the portal → Settings → Store
2. Under Custom Stores, add this repo URL
3. Click Sync
4. Apps appear under the Global App Store tab
