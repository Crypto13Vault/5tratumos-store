# Crypto13Vault 5tratumOS App Store

## Bitaxe Dashboard

Monitor, control, and auto-discover your Bitaxe ASIC fleet.

- **Auto-detect** — scans your LAN for Bitaxe miners
- **Benchmark** — hill-climbing frequency/voltage profiling
- **Control** — dynamic throttling to balance hashrate vs temperature
- **Plan** — 24h predictive planning using weather forecasts
- **History** — telemetry logging with SQLite

## Install

Add this store to your 5tratumOS:

1. **Settings → App Store**
2. Paste the GitHub repo URL
3. Install **Bitaxe Dashboard** from Community Stores
